In this section we will create an account with pyhtonanywhere that is a python ide and a cloud hosting service that allows you to host websites for free, we will also see the features of pythonanywhere

We will be working with pyhtonanywhere which is an online IDE

There are a few different subscription plans at pythonanywhere but for our project, the free one would do. We will first start by creating our Pythonanywhere account or login if you already have one, We can go to Pythonanywhere either by searching on google or I am also attaching a link below:

https://www.pythonanywhere.com/login/?next=/


If you are new to pythonanywhere click on signup and you will be redirected to a page looking like this

As I already mentioned that a beginner account would do for our project so we click on create a free beginner account and a screen like this will appear

After you have created the account you will get steps of pythonanywhere and basic guidelines of the UI of pythonanywhere read through the steps once, I will also explain the fields again

Console-In this part you can see the active consoles of python bash or SQL

Files-This place you will see the running files of your project, with the free version you can only work with one project at a time

Notebooks- you cant access this with a free account

Web apps- in this place you can see the running web apps

Now we will create our first web app, click on the pythonanywhere logo and go to the home page and from that click on the WEB tab on the top right

When you click on the web tab you will be redirected to this site-


Click on next and there will be list of microweb frameworks, select flask and after that, there will be a list of different python versions and select python 3.9 or whichever is the latest version. After you select the version of python,this page will appear:

This page is asking you where you want to put your code. Code on PythonAnywhere is stored in your home directory, /home/ yourusername, and in its subdirectories. Now, Flask is a particularly lightweight framework, and you can write a simple Flask app in a single file. PythonAnywhere is asking you where it should create a directory and put a single file with a really simple website. The default should be fine; it will create a subdirectory of your home directory calledmy site and thewill put the Flask code into a file called flask_app.py inside that directory.

After this configuration page for this site will appear :

On the top left section of the page Is the hostname for this website and add a web app (only for paid accounts) as we are using the free version we have to log in once in three months and click on Run until 3 months and the website is good to run.
